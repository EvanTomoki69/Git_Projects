La RAE define como computadora a cualquier aparato para llevar a cabo calculos matematicos, pues hasta una calculadora hace la funcion de COMPUTAR (calculos matematicos)

Diferencias entre hardware y software 
El hardware es aquello que podemos tocar con nuestras manos, una computadora posee diferentes piezas de hardware
Para que una computadora funcione minimamente debera contar con las siguientes piezas:
Un CPU con graficos integrados (Celeron, Core, Ryzen, Athlon)
Enfriamiento del CPU (Por aire o liquido)
Un stick de RAM (DDR3, DDR4)
Una tarjeta madre (ATX, mATX, ITX)
Memoria ROM (stick USB, o disco duro)
Fuente de alimentacion

Estos son los componentes MINIMOS para hacer que una PC funcione, aunque le puedes agregar otras como un gabinete (PC Case), ventiladores extra, una tarjeta de video dedicada, almacenamiento SSD SATA o NVME, mas discos duros, un lector y quemador de DVD, enfriamiento liquido, sensores, entre otras cosas.
Ahora bien, hay componentes externos que permiten interactuar con la computadora y ver lo que sucede dentro de ella, por ejemplo
Dispositivos de entrada y salida (I/O) o perifericos
